# Daily Quote iOS App

A clean and minimal iOS application built using SwiftUI and MVVM architecture.  
The app displays daily inspirational quotes, allows users to save favorites with persistence, and share quotes using native iOS share functionality.

---

# Features
-  Fetches random inspirational quotes
-  Save and remove favorite quotes
- Favorites persist across app restarts
-  Native iOS share sheet integration
-  Clean MVVM architecture
-  Reusable SwiftUI components

---

# Architecture
- SwiftUI for UI
- MVVM pattern
- `QuoteService` for data fetching
- `FavoritesStore` for persistence using `UserDefaults`
- Reusable `QuoteCardView`

---

# Screens
- Today: View a random quote and interact with it
- Favorites: Manage saved quotes with remove & share support

---

# Requirements
- iOS 17+
- Xcode 15+
- Swift 5.9+
